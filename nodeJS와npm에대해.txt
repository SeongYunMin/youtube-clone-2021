노드js는 JS를 브라우저에서 쓰는걸 넘어서 독립적으로 사용할 수 있게 해주는 프로그램이다
여타 파이썬이나 C++ 처럼 독립적으로 프로그래밍 가능하게끔 한다 이걸로 데스크탑 앱도 만들
수 있고 react native를 사용해 ios android 앱도 만들 수 있다
npm은 JS 언어를 위한 패키지 매니저이다 무슨말이냐면 npm과 노드js는 같이 써야하는 것이다
npm은 노드js가 상호작용할 수 있게끔 해주는 매니저이다 기본적으로 노드JS 설치할때 npm도
같이 설치된다 yarn 이라는 패키지 매니저인데 우리는 npm을 쓸 것이다
그럼 npm은 정확히 뭐란 말인가? 패키지매니저가 뭐야? 여러 똑똑한 사람들이 작은 패키지들을
만들었는데 우린 이 패키지들을 가져다가 쓸것이다 이것들은 매우 유용하기 때문
이 npm이 이 패키지들을 다운받을 수 있게 해준다 npm덕에 node.js가 커질 수 있다
npm은 누군가 이미 만들어논 패키지를 가져다 쓰기 너무 편하게 돼있다
npm 사이트 들어가서 다양한 패키지들을 찾아볼 수 있다 예를들어 express등의 유명한 패키지
드들이 있다 앞으로 이 npm을 통해 사람들이 만든 유용한 패키지들을 갖다 쓸 것이다
